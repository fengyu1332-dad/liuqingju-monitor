## 1. 功能概述

搜索功能支持搜索用户、论坛帖子、学习资源和悬赏项目，提供即时搜索浮层和完整搜索页面两种访问方式。

### 1.1 核心功能

- 支持四种搜索类型：用户、帖子、资源、悬赏
- 即时搜索（输入时自动搜索）
- 搜索结果分组展示
- 支持跳转到完整搜索页面
- 防抖处理（避免频繁搜索）

### 1.2 设计原则

- **即时反馈**：输入时实时显示搜索结果
- **分类清晰**：按类型分组展示结果
- **防抖优化**：300ms 延迟避免频繁搜索
- **大小写不敏感**：搜索关键词不区分大小写
- **模糊匹配**：支持部分匹配

---

## 2. 搜索范围

| 类型 | type值 | 搜索字段 |
|------|--------|---------|
| 用户 | user | 昵称、邮箱 |
| 论坛帖子 | post | 标题、内容摘要 |
| 学习资源 | resource | 标题、描述、标签 |
| 悬赏项目 | bounty | 标题、描述 |

---

## 3. SearchService 服务

### 3.1 核心方法

```javascript
const SearchService = {
  // 搜索用户
  searchUsers(keyword),

  // 搜索帖子
  searchPosts(keyword),

  // 搜索资源
  searchResources(keyword),

  // 搜索悬赏
  searchBounties(keyword),

  // 搜索全部
  searchAll(keyword),

  // 搜索指定类型
  searchByType(keyword, type),

  // 匹配工具函数
  matchKeyword(text, keyword)
}
```

### 3.2 搜索匹配规则

- **大小写不敏感**：keyword 转换为小写匹配
- **模糊匹配**：只要包含关键词即匹配
- **多词支持**：可选，支持按空格分割多个关键词（本轮先不实现）

### 3.3 搜索结果结构

```javascript
// searchAll 返回的结果结构
{
  users: [...],
  posts: [...],
  resources: [...],
  bounties: [...]
}
```

## 4. 用户界面

### 4.1 顶部搜索框

**位置**：导航栏中央或右侧

**结构**：
```
┌─────────────────────────────────────┐
│ 🔍 搜索内容...              [✕]      │ ← 搜索输入框
├─────────────────────────────────────┤
│ [全部] [用户] [帖子] [资源] [悬赏]  │ ← 类型筛选
├─────────────────────────────────────┤
│ 用户                    [查看全部]  │
│ ┌─────────────────────────────────┐ │
│ │ 👤 张三                        │ │
│ │ zhangsan@example.com           │ │
│ └─────────────────────────────────┘ │
│ 帖子                    [查看全部]  │
│ ┌─────────────────────────────────┐ │
│ │ 📝 IB选课建议...               │ │
│ │ 作者：李四                    │ │
│ └─────────────────────────────────┘ │
│ [查看完整搜索结果 →]              │
└─────────────────────────────────────┘
```

**功能**：
- 输入关键词即时触发搜索（300ms防抖）
- 按类型标签筛选结果
- 显示各类别前3条结果
- 点击"查看全部"跳转到 search.html
- 点击"查看完整搜索结果"跳转到 search.html

### 4.2 完整搜索页

**路由**：`search.html`

**URL参数**：`search.html?q=关键词&type=类型`

**布局**：
```
┌─────────────────────────────────────┐
│ 🔍 [搜索关键词...]              [Go] │
│ [全部] [用户] [帖子] [资源] [悬赏]  │
├─────────────────────────────────────┤
│ 搜索结果 (32)                        │
├─────────────────────────────────────┤
│ 用户 (8)                            │
│ ┌─────────────────────────────────┐ │
│ │ 👤 张三  email...  [查看]       │ │
│ │ ...                            │ │
│ └─────────────────────────────────┘ │
│ 帖子 (15)                           │
│ ┌─────────────────────────────────┐ │
│ │ 📝 标题...  作者...  [查看]    │ │
│ │ ...                            │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**功能**：
- 顶部搜索框（重复使用）
- 类型筛选
- 显示所有匹配结果
- 分页或无限滚动（可选，本轮不实现）

## 6. 注意事项

1. **防抖处理**：搜索输入延迟300ms执行，避免频繁搜索
2. **空搜索**：清空搜索内容时隐藏浮层
3. **点击外部关闭**：点击搜索浮层外部区域关闭浮层
4. **关键词高亮**：可选，搜索结果中高亮匹配关键词
5. **搜索历史**：可选，记录搜索历史（本轮不实现）
