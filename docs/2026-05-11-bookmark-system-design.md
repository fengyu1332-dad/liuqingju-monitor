**日期**: 2026-05-11
**版本**: v1.0
**状态**: 已批准

---

## 1. 功能概述

收藏功能允许用户收藏论坛帖子、学习资源和悬赏项目，支持在个人主页统一管理和在各页面侧边栏快捷访问。

### 1.1 核心功能

- 支持收藏帖子、资源、悬赏三种类型
- 点击切换收藏状态
- 个人主页统一管理
- 各页面侧边栏快捷入口

### 1.2 设计原则

- **即时反馈**：点击即切换，无需确认
- **视觉清晰**：空心星标(未收藏) / 实心星标(已收藏)
- **轻量化**：基于 localStorage，无需后端支持
- **可扩展**：预留其他类型扩展接口

## 2. 数据结构

### 2.1 收藏记录

```javascript
// localStorage key: 'liuqingju_bookmarks'
{
  userId: "user_xxx",
  bookmarks: [
    {
      id: "bm_xxx",
      type: "post",           // post | resource | bounty
      targetId: "post_xxx",   // 关联内容ID
      createdAt: "ISO日期"
    }
  ]
}
```

### 2.2 收藏类型

| 类型 | type值 | 说明 |
|------|--------|------|
| 论坛帖子 | post | 论坛板块的帖子内容 |
| 学习资源 | resource | 用户上传的学习资料 |
| 悬赏项目 | bounty | 悬赏广场的悬赏任务 |

## 3. BookmarkService 服务

### 3.1 核心方法

```javascript
const BookmarkService = {
  // 生成唯一ID
  generateId(),

  // 添加收藏
  addBookmark(userId, type, targetId),

  // 移除收藏
  removeBookmark(userId, targetId),

  // 检查是否已收藏
  isBookmarked(userId, targetId),

  // 获取用户所有收藏
  getBookmarks(userId),

  // 按类型获取收藏
  getBookmarksByType(userId, type),

  // 切换收藏状态
  toggleBookmark(userId, type, targetId),

  // 获取收藏详情（包含关联内容）
  getBookmarkWithDetails(userId, type)
}
```

### 3.2 事件

- `bookmarkAdded` - 收藏添加事件
- `bookmarkRemoved` - 收藏移除事件

## 4. 用户界面

### 4.1 收藏按钮

**位置**：帖子卡片、资源卡片、悬赏卡片的操作区域

**状态**：
- 未收藏：空心星标 ☆（灰色）
- 已收藏：实心星标 ★（金色/橙色）

**交互**：
- 登录用户：点击直接切换状态
- 未登录用户：点击弹出登录提示

### 4.2 个人主页收藏页

**路由**：`profile.html#bookmarks`

**功能**：
- 分类筛选（全部/帖子/资源/悬赏）
- 按时间倒序显示
- 显示收藏内容摘要
- 支持取消收藏
- 点击跳转到对应详情页

**布局**：
```
┌─────────────────────────────────────┐
│ 我的收藏                      (12)  │
├─────────────────────────────────────┤
│ [全部] [帖子] [资源] [悬赏]         │
├─────────────────────────────────────┤
│ ┌─────────────────────────────────┐ │
│ │ 📝 帖子：IB选课建议              │ │
│ │    收藏于 2小时前    [取消收藏] │ │
│ └─────────────────────────────────┘ │
│ ┌─────────────────────────────────┐ │
│ │ 📚 资源：Past Paper 2023        │ │
│ │    收藏于 1天前      [取消收藏] │ │
│ └─────────────────────────────────┘ │
│ ┌─────────────────────────────────┐ │
│ │ 💰 悬赏：申请文书修改            │ │
│ │    收藏于 3天前      [取消收藏] │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

### 4.3 侧边栏快捷入口

**位置**：论坛、资源、悬赏等页面右侧边栏

**显示内容**：
- 标题："我的收藏"
- 最近收藏（最多5条）
- 点击"查看全部"跳转个人主页

## 6. 注意事项

1. **登录检查**：未登录用户点击收藏时提示登录
2. **状态同步**：收藏状态变化时实时更新按钮样式
3. **内容删除**：当被收藏的原内容被删除时，收藏记录保留但显示"内容已不存在"
4. **存储限制**：localStorage 有大小限制，需要定期清理（保留最近500条）
