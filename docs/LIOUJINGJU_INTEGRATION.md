# 留情局情报监控系统集成指南

**版本**: v2.0  
**更新日期**: 2026-05-11

---

## 📋 概述

本指南说明如何将 university-monitor 监控系统集成到留情局论坛中，实现：
- AI 自动采集全球顶尖大学招生信息
- 每天 2 次自动更新（8:00 和 20:00 北京时间）
- 自动同步到留情局情报中心
- 管理员可在后台管理监控状态

## 🚀 部署步骤

### 第一步：创建 Supabase 项目

1. 访问 https://supabase.com 创建新项目
2. 获取以下凭证：
   - **Project URL**: `https://xxx.supabase.co`
   - **Service Role Key**: 在 Settings → API 中获取

### 第二步：初始化数据库

1. 进入 Supabase SQL Editor
2. 先执行 `supabase/tables.sql`（基础表）
3. 再执行 `supabase/intelligence_tables.sql`（留情局情报表）

### 第三步：创建 GitHub 仓库

1. 在 GitHub 创建新仓库 `university-monitor`
2. 上传 `backend/` 目录所有文件

### 第四步：配置 GitHub Secrets

| Secret 名称 | 值 |
|------------|-----|
| `DEEPSEEK_API_KEY` | DeepSeek API Key |
| `SUPABASE_URL` | Supabase Project URL |
| `SUPABASE_KEY` | Supabase Service Role Key |
| `BOT_USER_ID` | 机器人用户 UUID |

### 第五步：集成到留情局

#### 5.1 添加 JS 文件

复制以下文件到留情局项目：
- `js/intelligence-sync-service.js`
- `js/intelligence-service.js`

#### 5.2 在情报中心引入

在 `intelligence.html` 中添加：

```html
<script src="js/intelligence-sync-service.js"></script>
```

初始化同步：

```javascript
const syncService = new IntelligenceSyncService(
    'YOUR_SUPABASE_URL',
    'YOUR_SUPABASE_KEY'
);
syncService.startAutoSync();
syncService.manualSync();
```

## 🔄 工作流程

```
1. GitHub Actions 触发（每天 8:00, 20:00 UTC）
   ↓
2. 爬虫抓取 10 所大学官网
   ↓
3. DeepSeek AI 提取招生信息
   ↓
4. AI 双重验证（置信度 > 0.7）
   ↓
5. 数据格式转换（adapter.py）
   ↓
6. 推送到 Supabase（sync_service.py）
   ↓
7. 留情局前端拉取数据
   ↓
8. 更新情报中心展示
```

## 🛠️ 故障排查

### 常见问题

**Q: 爬虫被封禁？**  
增加请求间隔：
```yaml
monitor:
  request_delay: 10
```

**Q: 数据未同步？**
```javascript
await syncService.manualSync();
console.log(syncService.getStats());
```
