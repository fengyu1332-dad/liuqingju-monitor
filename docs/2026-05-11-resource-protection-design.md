**日期**: 2026-05-11

**状态**: 已批准

## 1. 功能概述

资源保护模块是数字资源管理的第三阶段，包含下载限制、积分门槛和防盗链三大功能。

### 1.1 核心功能

- **下载限制**：每日/每周下载次数限制，防止滥用
- **积分门槛**：设置资源最低积分要求，确保贡献者权益
- **防盗链**：防止外站直接引用资源链接

### 1.2 设计原则

- **公平使用**：合理限制下载，保证资源可用性
- **贡献优先**：鼓励用户贡献，保护贡献者权益
- **安全防护**：防止资源被恶意盗用

## 2. 下载限制

### 2.1 限制规则

| 限制类型 | 默认值 | 说明 |
|---------|-------|------|
| 每日下载上限 | 10次 | 24小时内最多下载10个资源 |
| 每周下载上限 | 50次 | 7天内最多下载50个资源 |
| 单个资源下载间隔 | 10分钟 | 同一资源10分钟内只能下载一次 |

### 2.2 数据结构

```javascript
// 用户数据增加下载记录
{
  id: "user_xxx",
  // ... 其他字段
  downloadStats: {
    dailyCount: 5,           // 今日下载次数
    weeklyCount: 23,         // 本周下载次数
    dailyResetTime: "ISO日期", // 今日重置时间
    weeklyResetTime: "ISO日期", // 本周重置时间
    recentDownloads: [        // 最近下载记录
      { resourceId, downloadedAt }
    ]
  }
}
```

### 2.3 下载流程

```
用户点击下载 → 检查登录状态
    ↓
检查下载限制
    ↓
├── 超过限制 → 提示等待或升级
└── 未超限制 → 扣除积分 → 下载资源
    ↓
更新下载记录
```

## 3. 积分门槛

### 3.1 门槛设置

| 资源类型 | 默认门槛 | 说明 |
|---------|---------|------|
| 普通资源 | 0积分 | 无门槛 |
| 优质资源 | 5积分 | 精选资源 |
| 稀缺资源 | 10积分 | 稀有资源 |
| 付费资源 | 自定义 | 用户设定 |

### 3.2 数据结构

```javascript
// 资源数据增加门槛字段
{
  id: "resource_xxx",
  // ... 其他字段
  accessLevel: {
    minPoints: 5,            // 最低积分要求
    level: "premium"         // basic/premium/rare/custom
  }
}
```

### 3.3 访问控制

```
用户访问资源 → 检查登录状态
    ↓
检查积分门槛
    ↓
├── 积分不足 → 提示获取积分途径
└── 积分足够 → 显示资源详情/下载按钮
```

## 4. 防盗链

### 4.1 防护策略

| 策略 | 说明 |
|------|------|
| 来源检查 | 检查请求来源是否为本站 |
| 时间戳验证 | 下载链接带有效期 |
| Token验证 | 下载链接带用户Token |
| 水印添加 | 资源内容添加水印 |

### 4.2 下载链接格式

```
/download/{resourceId}/{timestamp}/{token}
```

### 4.3 防护流程

```
请求下载链接 → 验证来源
    ↓
生成带Token的临时链接
    ↓
用户点击链接 → 验证Token和时间戳
    ↓
验证通过 → 返回资源文件
```

## 7. 注意事项

1. **时间同步**：下载限制需要考虑时区和时间重置
2. **积分返还**：下载失败时需返还积分
3. **白名单**：管理员可不受下载限制
4. **用户体验**：友好提示剩余下载次数和积分余额
