## 1. 功能概述

管理员后台提供完整的社区管理功能，支持内容管理、用户管理和数据统计。

### 1.1 核心功能

- **数据总览**：社区实时数据统计
- **内容管理**：帖子、资源、悬赏的审核/删除/置顶
- **用户管理**：用户列表、禁用/启用、设置管理员
- **权限控制**：仅管理员可访问

### 1.2 设计原则

- **权限隔离**：普通用户无法访问
- **操作安全**：删除等危险操作需要确认
- **实时反馈**：操作结果即时显示
- **轻量化**：基于现有 localStorage 数据

## 2. 管理员权限

### 2.1 管理员判断

```javascript
// 预设管理员邮箱列表（可在后台添加更多）
const ADMIN_USERS = [
    'admin@liuqingju.com',
    'admin@example.com'
];

function isAdmin() {
    const user = AuthService.getCurrentUser();
    return user && ADMIN_USERS.includes(user.email);
}

function isSuperAdmin() {
    const user = AuthService.getCurrentUser();
    return user && user.email === 'admin@liuqingju.com';
}
```

### 2.2 用户菜单入口

在 user-menu.js 的用户下拉菜单中：
- 普通用户：不显示"管理后台"
- 管理员用户：显示"管理后台"菜单项

## 5. 用户管理模块

### 5.1 用户列表

**功能列表**：
- 查看用户列表
- 禁用/启用用户
- 设置/取消管理员
- 按状态筛选（全部/正常/禁用）

**列表字段**：

| 贡献值 | 贡献值 |
| 注册时间 | 注册日期 |

| 操作 | 禁用/管理员按钮 |

### 5.2 用户操作

| 操作 | 说明 |
|------|------|
| 禁用用户 | 用户无法登录，内容保留 |
| 启用用户 | 恢复用户登录权限 |
| 设置管理员 | 添加到管理员列表 |
| 取消管理员 | 从管理员列表移除 |

## 6. AdminService 服务

### 6.1 核心方法

```javascript
const AdminService = {
    // 权限检查
    isAdmin(),
    isSuperAdmin(),

    // 数据统计
    getStats(),

    // 内容管理
    deletePost(postId),
    pinPost(postId),
    unpinPost(postId),
    deleteResource(resourceId),
    featureResource(resourceId),
    deleteBounty(bountyId),
    closeBounty(bountyId),

    // 用户管理
    disableUser(userId),
    enableUser(userId),
    setAdmin(userId),
    removeAdmin(userId),
    getAllUsers(),
    getUserById(userId)
}
```

## 8. 注意事项

1. **权限保护**：未登录或非管理员用户访问 admin.html 时跳转首页
2. **操作确认**：删除等操作需要确认弹窗
3. **操作日志**：记录管理员关键操作（可选）
4. **数据一致性**：删除内容时同步更新相关统计
