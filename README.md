## 📋 快速部署（预计 30 分钟）

### 第一步：在 GitHub 创建仓库

1. 访问 https://github.com/new
2. 创建新仓库 `liuqingju-monitor`
3. 上传 `university-monitor-package` 目录下的所有文件

### 第二步：配置 GitHub Secrets

在仓库 **Settings → Secrets and variables → Actions** 中添加：

| Secret 名称 | 值 | 说明 |
|------------|-----|------|
| `DEEPSEEK_API_KEY` | `sk-xxxx...` | DeepSeek API Key |
| `SUPABASE_URL` | `https://xxx.supabase.co` | Supabase 项目 URL |
| `SUPABASE_KEY` | `eyJhbGci...` | Supabase Service Role Key |
| `BOT_USER_ID` | `xxx-xxx...` | 留情局机器人用户 ID |

### 第三步：创建 Supabase 项目

1. 访问 https://supabase.com
2. 创建新项目
3. 在 **SQL Editor** 中执行 `supabase/init-database.sql`

### 第四步：触发首次运行

1. 进入 GitHub 仓库 **Actions** 页面
2. 选择 **"University Admission Monitor"**
3. 点击 **"Run workflow"**
4. 等待运行完成

## ⚙️ 工作原理

```
GitHub Actions (每天 8:00, 20:00 UTC)
    ↓
爬虫抓取 10 所大学官网
    ↓
DeepSeek AI 提取招生信息
    ↓
AI 双重验证（置信度 > 0.7）
    ↓
数据格式转换 (adapter.py)
    ↓
推送到 Supabase (sync_service.py)
    ↓
留情局前端拉取数据 (intelligence-sync-service.js)
    ↓
情报中心展示
```
